## Detailed description

```
Write in your language. Be understood anywhere.

WorldWrite is an inline communication assistant that helps you write, translate, polish, and understand messages across languages — right inside any text field in your browser. It uses your own AI API key, and nothing is stored.

HOW IT WORKS
1. Select text in any input or message.
2. Right-click → WorldWrite, or use a keyboard shortcut.
3. Review the result in a clean preview card.
4. Click Replace to use it — nothing changes until you do.

FOR TEXT YOU WROTE (replaces after you approve)
• Translate your text into another language
• Polish your writing to sound natural and fluent
• Make it more professional for work and formal messages
• Custom rewrite with your own instruction

FOR A MESSAGE YOU RECEIVED (read-only, never changes the page)
• Explain what a message means, including tone and nuance
• Translate it into your own language

REFINE ANYTHING
• One-click tweaks: Shorter, Warmer, More formal, More casual, Simpler
• Or type a follow-up instruction to iterate
• Optional word-level "Show changes" diff

PRIVATE BY DESIGN
• Only the text you select is sent — never the page URL, page title, surrounding content, or browsing history.
• Your text and the AI results are never stored.
• Your API key is used only by the extension's background and is sent only to the AI provider you configure. It is never written to the page, never logged, and never included in exported settings.
• Replacement only happens after you explicitly click Replace.

BRING YOUR OWN AI KEY
Supported providers: OpenAI, Anthropic (Claude), Google Gemini, OpenRouter, and DeepSeek — with more coming soon. You can also point it at any OpenAI-compatible endpoint.

OPEN SOURCE
WorldWrite is open source (MIT). Review the code, file issues, or contribute:
https://github.com/JerryGgzm/WorldWrite

No account. No tracking. No data collection. Just your writing, your key, your privacy.
```
